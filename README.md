# NI-MLP Semestral Work

Developed models to predict the affinity be-
tween misconceptions and incorrect answer choices in
multiple-choice mathematics questions. Implemented
a similarity search using a fine-tuned BGE embedding model
and zero-shot learning techniques to identify the cor-
rect misconceptions.

## How to read the work

It is recommend to real only the `html` files starting by reading the `index.html` file which will lead you through the work and what has been accomplished.
